# 📚 Cursos Apuntes

Aplicación Android para guardar y organizar tus apuntes de estudio de forma clara y ordenada.

---

## ¿Qué puedes hacer con esta app?

### 📁 Organiza por secciones
Crea secciones para cada curso o tema que estudies, por ejemplo "Matemáticas", "Programación" o "Inglés". Cada sección guarda todos los apuntes relacionados a ese tema en un solo lugar.

### 📝 Guarda tus apuntes
Dentro de cada sección puedes agregar apuntes. Cada apunte tiene tres campos:
- **Función**: el nombre o título del apunte
- **Para qué sirve**: una descripción de qué hace o para qué se usa
- **Ejemplo**: código u otro ejemplo práctico (opcional)

### 🔍 Busca rápido
Usa la barra de búsqueda para encontrar cualquier sección o apunte sin tener que revisar todo manualmente.

### ✏️ Edita y elimina
¿Te equivocaste en algo? Toca el ícono ⋮ en cualquier tarjeta para editarla o eliminarla.

### ☑️ Selección múltiple
Mantén presionada una tarjeta para activar el modo selección. Luego puedes seleccionar varios apuntes o secciones a la vez para eliminarlos o compartirlos juntos.

### 🔀 Cambia el orden
Usa las flechas ↑↓ que aparecen en cada tarjeta para subir o bajar su posición en la lista y organizarlas como prefieras.

### 📤 Comparte tus apuntes
Desde el detalle de un apunte toca el ícono de compartir para enviarlo por WhatsApp, Telegram, Gmail o cualquier app de mensajería con el texto ya formateado.

### 💾 Guarda una copia de seguridad
Desde Ajustes puedes exportar todos tus apuntes a un archivo en la carpeta Descargas de tu celular. Así si desinstallas la app no pierdes nada.

### 🔄 Restaura tus datos
Si reinstalaste la app o cambiaste de celular, puedes importar el archivo de copia de seguridad y recuperar todo exactamente como lo dejaste.

### 🌙 Modo oscuro
Activa el modo oscuro desde Ajustes para cuidar tu vista cuando estudias de noche.

---

## ¿Cómo se guarda todo?

Todo se guarda automáticamente en tu celular. No necesitas internet ni crear una cuenta. Tus apuntes son solo tuyos.

---

## Requisitos

- Celular Android con versión 11 o superior
